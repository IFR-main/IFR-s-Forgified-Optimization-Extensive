
# 简介

这是一个使用了**Cleanroom**的整合包。

此整合包集成了1.12.2的一些优化模组，使得低门槛设备在1.12.2享受目前很好的优化。

# 使用说明

使用前请先安装**Java 25**和**Java 8**。

注：请不要更新**Cleanroom 模组式启动器非官方版** 和 **Fugue** 模组版本，否则可能冲突。

首次启动时，**Cleanroom 模组式启动器非官方版** 模组会弹出安装Cleanroom界面，请按照说明安装Cleanroom。

首次安装可能时间较长。

此整合包提供了**Optifine**和**Nothirium**。
